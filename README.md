## 📖 故事背景
> *「十六年前，一場叛亂與大火改變了一切。」*

公爵府的嬰兒在混亂中被調換，真正的千金流落民間。  
如今，妳被接回羅維恩公爵府。然而，府中那位被寵愛了十六年的假千金——**莉薇安**，仍然站在權力的中心。

在王權交替的時刻，兩位千金被推上同一座舞台。  
血統、權力與人心交織，妳的每一步，都將刻下命運。

### 👥 命運的交織者
* **莉薇安** —— 優雅完美的假千金，深受家族與貴族喜愛。
* **皇太子 艾德里安** —— 難以捉補、如薄冰般寒冷的王國繼承人。
* **騎士 羅倫斯** —— 寡言而忠誠，誓死守護妳的王國騎士。

本版本新增存檔功能

本遊戲的存檔是儲存在你目前的瀏覽器緩存中，**不同設備不通用**，也就是說如果在手機上玩，存檔就不會出現在電腦上。如果執行了「清除瀏覽器紀錄」或「清空快取」，所有的存檔資料將會被永久刪除，連神都救不回來，另外，無痕模式下也不會保存資料。

如果發現存檔壞了，別擔心，這是公爵府的時空扭曲（其實是bug）。可以按 Ctrl + F5 (電腦)，或重新開啟分頁，通常能解決99%的魔法問題◐∇◐*

存檔系統目前處於測試階段。雖然我已經抓過很多次蟲了，但難保還是會有漏網之魚。若您在冒險途中遇到任何異常，再麻煩回報給我，我會帶著除蟲噴霧，在某個深夜痛苦的修改程式碼´•ᴗ•

### 📜 畫室誓約（使用規則）

| 章節 | 誓約內容 | 塵世解讀 (規則) |
| :--- | :--- | :--- |
| **Ⅰ. 噤聲與低語** | 「在此流動的應是靈魂的低語，而非惡意的噪音。凡汙穢、咒罵或跨越國境的狂言，都將被陰影抹除。」 | 禁止謾罵、人身攻擊或不當言論。 |
| **Ⅱ. 記憶的定錨** | 「每道筆跡都是一段封存的時光。請勿挪動他人的記憶。若妳隨意撥亂命運的經緯，妳也將在迷霧中失去歸處。」 | **嚴禁刪除他人留言**，請尊重創作。 |
| **Ⅲ. 最後的遺產** | 「請維持這片祭壇的優雅，畢竟，這可能是我們最後的遺產。」 | 請保持畫布美觀，請勿惡意塗鴉。 |
| **Ⅳ. 禁名** | 「影子是不需要名字的。」 | 請勿留下真實個資，建議使用代號。 |
| **Ⅴ. 印記** | 「我們是共謀者，而非孤島。」 | 歡迎使用小貼紙或文字與他人互動。 |
