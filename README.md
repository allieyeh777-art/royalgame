## 📖 故事背景
> *「十六年前，一場叛亂與大火改變了一切。」*

公爵府的嬰兒在混亂中被調換，真正的千金流落民間。  
如今，妳被接回羅維恩公爵府。然而，府中那位被寵愛了十六年的假千金——**莉薇安**，仍然站在權力的中心。

在王權交替的時刻，兩位千金被推上同一座舞台。  
血統、權力與人心交織，妳的每一步，都將刻下命運。

### 👥 命運的交織者
* **莉薇安** —— 優雅完美的假千金，深受家族與貴族喜愛。
* **皇太子 艾德里安** —— 難以捉補、如薄冰般寒冷的王國繼承人。
* **騎士 羅倫斯** —— 寡言而忠誠，誓死守護妳的王國騎士。

### 📜 畫室誓約（使用規則）

| 章節 | 誓約內容 | 塵世解讀 (規則) |
| :--- | :--- | :--- |
| **Ⅰ. 噤聲** | 「在此流動的應是靈魂的低語。」 | 禁止謾罵、人身攻擊或不當言論。 |
| **Ⅱ. 定錨** | 「請勿挪動他人的記憶。」 | **嚴禁刪除他人留言**，請尊重創作。 |
| **Ⅲ. 遺產** | 「維持這片祭壇的優雅。」 | 保持畫布美觀，請勿惡意塗鴉。 |
| **Ⅳ. 禁名** | 「影子是不需要名字的。」 | 請勿留下真實個資，建議使用代號。 |
| **Ⅴ. 印記** | 「我們是共謀者，而非孤島。」 | 歡迎使用小貼紙或文字與他人互動。 |
