# 冠冕

### 📖 故事背景
十六年前，一場叛亂與大火改變了一切。
公爵府的嬰兒在混亂中被調換，真正的千金流落民間。

如今，你被接回羅維恩公爵府。
然而，府中那位被寵愛了十六年的「千金小姐」——莉薇安，仍然站在所有人的視線中心。

在王國即將選出未來王后的時刻，兩位「公爵千金」被推上同一座舞台。
血統、權力與人心交織的宮廷之中，每一步都可能改變命運。

你將遇見：

莉薇安 —— 優雅完美的假千金，深受家族與貴族喜愛。

皇太子艾德里安 —— 難以捉摸的王國繼承人。

騎士羅倫斯 —— 寡言而忠誠的王國騎士。

在這場無聲的權力角逐中，
你會成為王后、逃離宮廷，還是走向另一條命運？
